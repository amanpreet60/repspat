How to use

create a working folder

go to 'https://github.com/amanpreet60/repspat/releases/tag/Initial'
download the 'repspat-0.1.0-py3-none-any.whl' file on your working folder

create an .venv

run in terminal:
pip install repspat-0.1.0-py3-none-any.whl

download a.ipynb file from github to the same local folder
https://github.com/amanpreet60/repspat/blob/main/a.ipynb

run the file
